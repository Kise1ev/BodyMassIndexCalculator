<h3>This program is a simple body mass index (BMI) calculator. It allows the user to enter their height in centimeters and weight in kilograms, and then calculates and displays the BMI value and the user's status based on this value.</h3>
<h3>BMI is a numerical measure of body mass to height ratio. It's one way of determining whether a person is underweight, normal or overweight.</h3>
